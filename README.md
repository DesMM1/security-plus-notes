Study notes and concept breakdowns while preparing for the **CompTIA Security+ (SY0-701)** certification.

Every domain has its own file. Notes are written in my own words as I work through the material — not copy-pasted from transcripts. Writing it out is how it sticks.

- [ ] Domain 2 — Threats, Vulnerabilities, and Mitigations (22%)
- [ ] Domain 3 — Security Architecture (18%)
- [ ] Domain 4 — Security Operations (28%)

## Study Approach

1. Work through one domain at a time, start to finish
2. Write notes in my own words — no copy-paste
3. Hit each concept from multiple angles (reading, video, practice questions)
4. Apply concepts in the home lab where possible
5. Take practice exams only after the domain is written up

Primary resources: Professor Messer SY0-701 course, CompTIA official study guide, TryHackMe for hands-on reinforcement.
